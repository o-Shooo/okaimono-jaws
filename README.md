# お買い物じょーず (okaimono-jaws)

日々の食費・日用品の購入を記録・管理するスマートフォンアプリ。買い物リストを作成し、購入時に店舗・日付・数量・価格を記録することで、いつ・どこで・いくらで買ったかを把握できる。

## 機能

### MVP
- 商品マスタ登録（手入力）
- 買い物リスト作成・管理
- 店舗マスタ（選択式・随時追加）
- 購入記録（店舗・日付・数量・価格）
- 購入履歴閲覧

### 将来機能
- レシート撮影（OCR）で購入情報を自動入力
- バーコードスキャンで商品登録

## 技術スタック

| 分類 | 技術 |
|------|------|
| フレームワーク | Expo SDK 55 + Expo Router v55 |
| 言語 | TypeScript |
| 状態管理 | Zustand |
| バックエンド | Supabase |
| パッケージマネージャー | Bun |
| 対応OS | iOS / Android |

## ディレクトリ構成

```
src/
  app/          # Expo Router ルート（ファイルベースルーティング）
  components/   # 再利用可能なUIコンポーネント
  constants/    # 定数・テーマ
  hooks/        # カスタムフック
assets/         # 画像・フォント等の静的ファイル
```

## 開発環境セットアップ

```bash
bun install
bun start
```
